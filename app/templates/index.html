About This Project

This web application is designed to streamline attendance management in educational institutions using face recognition technology. It supports multiple user roles (Super Admin, Teacher, Student) and provides a secure, user-friendly interface for managing attendance records.

  • Super Admin: Add/manage teachers and students, view all attendance data.
  • Teacher: Mark attendance (face recognition/manual), manage students, view reports.
  • Student: Attendance is marked via face recognition; no login required.
Key Features
  • Face recognition-based and manual attendance marking
  • Role-based access for admins and teachers
  • Student management with photo upload and USN/semester tracking
  • Attendance reports (daily, monthly, student-wise)
  • Secure authentication and session management
  • Modern, responsive web interface
  • Powered by Python Flask, PostgreSQL, Bootstrap
How to Use
  1. Super Admin and Teachers can log in from this page.
  2. Admins add teachers and students; teachers manage their own students and mark attendance.
  3. Students are marked present via face recognition or manually by teachers.
